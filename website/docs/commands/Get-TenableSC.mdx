## SYNOPSIS

Generic "Get a Tenable SC object" call for those that do not need complex handling.

## SYNTAX

```powershell
Get-TenableSC [-Resource] <Object> [-Properties <Object>] [[-Key] <ValidateNotNullOrEmptyAttribute>]
 [<CommonParameters>]
```

## DESCRIPTION

Most calls in Tenable SC have a Get-TenableSC\<resource\> command which gets the specifics of that type.
For many types we can also use this function as most returned items do not need more than a type assignment of the results.
