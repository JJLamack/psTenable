## SYNOPSIS

Gets a list of Users from Tenable Security Center.
If CurrentUser flag is set then only gets the current user.

## SYNTAX

### Default (Default)

```powershell
Get-TenableSCUser [[-User] <Object>] [-Properties <Object>] [<CommonParameters>]
```

### Org

```powershell
Get-TenableSCUser [-Properties <Object>] -Organization <Object> [<CommonParameters>]
```

### CurrentUser

```powershell
Get-TenableSCUser [-Properties <Object>] [-CurrentUser] [<CommonParameters>]
```
